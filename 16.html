Лекция №7.1

1. Подключение

Обязательный CDN-скрипт:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Пример</title>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/3.2.1/anime.min.js"></script>
        </head>
        <body>
            <div id="target"></div>
        </body>
        </html>

2. Базовая анимация

Минимальный пример:

        anime({
            targets: '#box',
            translateX: '250px',
            duration: 1000,
            easing: 'easeInOutQuad'
        });

3. Timeline

Последовательность анимаций:

        const tl = anime.timeline();
         
        tl.add({
            targets: '.el',
            opacity: 0.5,
            duration: 500
        })
        .add({
            targets: '.el',
            translateY: 100
        });

4. Управление

Методы контроля:

        const anim = anime({
            targets: '.obj',
            rotate: '1turn'
        });
         
        anim.pause();  // Пауза
        anim.play();   // Запуск

5. Stagger

Каскадная анимация:

        anime({
            targets: '.item',
            scale: 1.2,
            delay: anime.stagger(100),
            duration: 500
        });

6. Ключевые параметры

Важные настройки:

        anime({
            loop: 3,          // Кол-во повторов
            direction: 'alternate',
            autoplay: false  // Ручной старт
        });

7. Анимация по пути SVG

Движение объекта вдоль SVG-пути:

        <svg width="200" height="200">
            <path id="motion-path" d="M10 80 Q 100 10 190 80" fill="none" stroke="grey"/>
            <circle id="mover" cx="0" cy="0" r="5" fill="red"/>
        </svg>
         
        anime({
            targets: '#mover',
            translateX: anime.path('#motion-path')('x'),
            translateY: anime.path('#motion-path')('y'),
            duration: 3000,
            loop: true
        });

8. Интерактивный переключатель

Анимация при клике с изменением состояния:

        document.querySelector('.switch').addEventListener('click', () => {
            anime({
                targets: '.toggle',
                translateX: '+=50px',
                backgroundColor: '#4CAF50',
                duration: 500,
                easing: 'spring(1, 80, 10, 0)'
            });
        });

9. Параметры анимации в реальном времени

Динамическое изменение свойств:

        const animation = anime({
            targets: '.dynamic',
            translateX: 100,
            update: function(anim) {
                console.log('Прогресс: ' + Math.round(anim.progress) + '%');
            },
            begin: function() {
                console.log('Старт анимации!');
            }
        });

10. Комплексная трансформация

Сочетание нескольких преобразований:

        anime({
            targets: '.complex',
            scale: [1, 1.5],
            rotate: '360deg',
            skew: '15deg',
            borderRadius: ['0%', '50%'],
            duration: 1500,
            direction: 'alternate'
        });

11. Кастомные easing-функции

Использование кривых Безье:

        anime({
            targets: '.custom-ease',
            translateY: 100,
            easing: 'cubicBezier(0.785, 0.135, 0.150, 0.860)',
            duration: 1000,
            loop: true
        });

12. Анимация текстового контента

Посимвольное появление текста:

        const textElement = document.querySelector('.text-anim');
        textElement.innerHTML = textElement.textContent
            .split('')
            .map((char, i) => `<span style="opacity:0">${char}</span>`)
            .join('');
         
        anime({
            targets: '.text-anim span',
            opacity: [0, 1],
            delay: anime.stagger(50),
            duration: 500
        });
