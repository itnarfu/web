1. Подключение библиотеки

Добавьте скрипт anime.js через CDN:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Пример</title>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/
         3.2.1/anime.min.js"></script>
        </head>
        <body>
            <div id="box"></div>
        </body>
        </html>

2. Базовая анимация

Перемещение элемента:

        anime({
            targets: '#box',
            translateX: '250px',
            duration: 1000,
            easing: 'easeInOutQuad'
        });

3. Цепочка анимаций

Использование timeline:

        const timeline = anime.timeline();
         
        timeline.add({
            targets: '.item',
            opacity: 0.5,
            duration: 500
        }).add({
            targets: '.item',
            translateY: 100,
            duration: 1000
        });

4. Управление анимацией

Методы контроля:

        const anim = anime({
            targets: '.element',
            translateX: 100
        });
         
        anim.pause();  // Пауза
        anim.play();   // Продолжить
        anim.restart();// С начала

5. Анимация нескольких элементов

Задержка между элементами:

        anime({
            targets: '.circle',
            scale: 1.5,
            delay: anime.stagger(100),
            duration: 500
        });

6. SVG-анимация

Рисование контура:

        anime({
            targets: '#path',
            strokeDashoffset: [anime.setDashoffset, 0],
            duration: 2000
        });
