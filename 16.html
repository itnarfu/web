Лекция №7.1

1. Подключение

Обязательный CDN-скрипт:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Пример</title>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/3.2.1/anime.min.js"></script>
        </head>
        <body>
            <div id="target"></div>
        </body>
        </html>

2. Базовая анимация

Минимальный пример:

        anime({
            targets: '#box',
            translateX: '250px',
            duration: 1000,
            easing: 'easeInOutQuad'
        });

3. Timeline

Последовательность анимаций:

        const tl = anime.timeline();
         
        tl.add({
            targets: '.el',
            opacity: 0.5,
            duration: 500
        })
        .add({
            targets: '.el',
            translateY: 100
        });

4. Управление

Методы контроля:

        const anim = anime({
            targets: '.obj',
            rotate: '1turn'
        });
         
        anim.pause();  // Пауза
        anim.play();   // Запуск

5. Stagger

Каскадная анимация:

        anime({
            targets: '.item',
            scale: 1.2,
            delay: anime.stagger(100),
            duration: 500
        });

6. Ключевые параметры

Важные настройки:

        anime({
            loop: 3,          // Кол-во повторов
            direction: 'alternate',
            autoplay: false  // Ручной старт
        });
